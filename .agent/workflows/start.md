# BondConnect Start Workflow

Follow these steps to initialize your environment and begin work on the first implementation task.

// turbo
1. Run the environment check script:
```bash
./scripts/check-env.sh
```

2. Review the core implementation rules in `.agent/agent-rules.md`.

3. Review the Master Index to understand the documentation structure:
`docs/technical/README.md`

4. Initialize the project infrastructure by following the BOND-01 specification:
`docs/technical/implementation/specs/BOND-01-firebase-setup.md`

5. Once BOND-01 is complete and verified, proceed to BOND-02.
